# Territory Registry

This registry tracks active and future topical territories.

Territory folders should be created when a territory is ready for active work. Keep exploratory ideas in this registry until they justify a brief and map.

## Active Territories

Core role:

- AI discovery.
- Local search.

## Promotion Criteria

A candidate territory becomes active when it has:

- Clear commercial relevance.
- Distinct buyer anxieties.
- Enough depth for multiple durable subclusters.
- A source base that can support evidence-led writing.
- A role in Off Piste's positioning.
- Existing content or planned investment worth organising.
